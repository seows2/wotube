# wotube

## 프로젝트 소개

유튜브를 클론하는 프로젝트로 포트폴리오 및 웹개발 능력을 올리기위해 제작했습니다. 로그인 기능, 동영상, 댓글 CRUD, DB를 mongo altas에 분리하고 크기가 큰 동영상을 외부에 저장할 수 있도록 AWS S3를 사용하였습니다.
